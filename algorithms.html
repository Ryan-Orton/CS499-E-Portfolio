Algorithms and Data Structures Enhancement

Artifact Description

The artifact I selected for this enhancement is my Weight Tracker mobile application, which I originally created in CS 360 Mobile Architecture and Programming. This application allows users to create an account, log in, set a goal weight, record weight entries, and store that information in a local SQLite database.

Why I Selected This Artifact

I selected this artifact for my ePortfolio because it already included a useful collection of weight-entry data, which made it a strong candidate for an algorithms and data structures enhancement.

Enhancement Performed

To enhance the artifact, I implemented a weight-analysis feature that processes the list of stored entries and calculates summary information including starting weight, current weight, total change, average weight, distance from goal, and overall trend. To do this, I created a helper class that copies the list, sorts the entries in chronological order, iterates through the data, computes averages, and compares earlier values to more recent values to classify the user's trend as gaining, losing, or stable.

Reflection

Through this process, I learned that even a simple list of stored data can become much more useful when it is processed intentionally. One challenge I faced was deciding how to design a trend algorithm that was simple enough for the current application but still meaningful and easy to explain.
